Windows Tortoise-Git ======================================================================================================================

1) Eine Repository auschecken:

Auf das Verzeichnis gehen mit rechter MausTaste Git Clone auswählen und z.B. http://github.conti.de/CTZS/GRID und den Ordner auf der Platte eintragen

2) Mit Änderungen soll in ein branch lokal eingecheckt werden:

Auf das Verzeichnis gehen mit rechter Maus, TortoiseGit -> Switch/Checkout auswählen, dann mit Create New Brach einen Branch erstellen, wahrscheinlich damit auch ausgewählt
Darüber kann auch der Branch ausgewählt werden, Vorsicht vorher den aktuellen branch commiten

Das lokale einchecken geshcieht mit Commit, Auf das Verzeichnis gehen mit rechter Maus -> Commit -> "ActualBranchName" auswählen

3) lokaler Branch soll auf Server kommen:

Ein Branch kann neu angelegt werden, wenn Auf das Verzeichnis gehen mit rechter MausTaste TortoiseGit -> push auswählen, remote branch entweder auswählen, oder neu reinschreiben

Danach werden alle Commits gepushed

4) Merge

Mand switched zum Master (TortoiseGit -> Switch/Checkout) und macht dann einen merge (TortoiseGit -> Merge), sucht den in From den Branch aus, der in master gemerged werden soll


5) Wenn Winowspasswort geändert:

Gehe zu Systemsteuerung->Benutzerkonten->Eigene Anmeldeinfornmationen verwalten: Dort git_hhtp://github.conti.de rauslöschen!!

Dann in Webpage wieder neu einloggen oder git gitTortoise
