0.12.4 Zertifizierung
 
Zur Info, falls das Problem auch bei Ihnen jemals aufteten sollte!

Folgendes ist zum Zertifizieren von Lotus Notes zu tun:

1) Lotus Notes starten
2) "Datei" --> "Extras" --> "Benutzer ID ..." wählen
3) Passwort eingeben
4) "Zertifikate" anklicken
5) "Zertifikat anfordern ..." wählen
6) "Adresse ..." wählen
7) "Global Addressbook" wählen
7) "$UserAdmins_fr" eingeben und dann im Fenster anklicken
8) auf den Knopf "An >" drücken
9) im rechten Auswahlfeld "$DomainAdmins" anwählen und "entfernen" klicken, so dass hier nur noch "$UserAdmins_fr" steht
10) auf "OK" klicken
11) auf "Senden" klicken
12) etwa 10 Minuten warten
13) es kommt eine eMail, mit Zertifikatinformationen. Öffnen!
14) "Aktionen" --> "Zertifikat annehmen ..." wählen

Gruß,

0.12.7 UMS-Funktion (Faxe per Notes empfangen und senden)

1) Anleitung für den Fax Versand mit Lotus Notes über Unified
Messaging
Wenn ihr Notes Client für den Fax Versand per Unified Messaging freigeschaltet
wurde, können Sie mit ihrem Notes Client Faxe verschicken bzw. empfangen.

Anleitung für Lotus Notes Version 6:

1. Erstellen Sie ein neues Memo, indem Sie in ihrem Notes Client auf den
entsprechenden Button klicken.
2. In das An/To: Feld tragen Sie die gewohnte Fax Nebenstelle ein (im Bild als
XXXX dargestellt), gefolgt von @FAXG3@FR_UMS
3. Die übrigen Felder können Sie mit beliebigem Text füllen. Sie können auch
Textanhänge wie gewohnt an das Memo hängen, die dann ebenfalls per Fax
versendet werden. Ebenso können zusätzliche Verteiler bzw. Einzeladressen
hinzugenommen werden.
4. Klicken Sie auf „Senden“ um das Fax abzuschicken.
5. Hinweis: Office Dokumente (Word, Excel, usw.) werden so versendet, wie sie
in der Vorschau von Microsoft Office dargestellt werden.
2) Anleitung für den Fax Empfang mit Lotus Notes über Unified Messaging
Damit Sie Faxe empfangen können, müssen Sie ihre UMS-Faxnummer bekannt
geben. Dazu verwenden Sie bitte die folgende Rufnummer:
+49 69 7603 88 XXXX
Beispiel:
Ihre Nebenstelle ist 0123, dann lautet ihre Fax-Nummer: +49 69 7603 88 0123
